# Brewery Buddy
This is an Android app I created that consumes a Brewery API (http://www.brewerydb.com/developers/docs)

# Search
Reactive search updates with service results as you type.  
Search by:

--brewery by name  
--beer by name  
--city


# Java
Libraries I am using  
--Dagger2  
--RxJava2  
--RxAndroid  
--Retrofit2  
--Butterknife
